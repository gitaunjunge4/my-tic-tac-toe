# INTRODUCTION

This is the repository to my Tic-Tac-Toe game. It was to help me improve on my coding and problem solving skills.

## OVERVIEW

### The challenge
Users should be able to :
-  See hover states for all interactive elements on the page
-  Play the game either solo vs the computer or multiplayer against another person
-   Have an interactive UI that enables the user to input their names
-  Keep tally of the score for both players as well as the draw
